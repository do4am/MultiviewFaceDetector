

- FaceDetection.py is the script for face detection
- NMS.py contain the non-maximal suppression.
- Data folder contains data of evaluation set and list files link to each image. This folder also contains the fully-convolutional version of AlexNet including network structure .prototxt files and its parameters .caffemodel
- Results folder contains output of the script
- Temp folder contains temp file while the code is running.
